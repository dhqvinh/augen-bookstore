# bookstore-frontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
Open a browser and enter this url: http://localhost:8080

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
